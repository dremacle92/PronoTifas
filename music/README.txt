MUSIQUE DE FOND — Prono Les Tifas
==================================

Dépose ici le fichier audio de la musique de fond, nommé EXACTEMENT :

    allumer-le-feu.mp3

Le site lit automatiquement « music/allumer-le-feu.mp3 » via le bouton 🎵
(en haut à gauche). Un clic lance / coupe la musique ; le choix est mémorisé
(au retour sur le site, la musique reprend dès le 1er clic/tap — le navigateur
interdit de démarrer le son sans une action de l'utilisateur).

Format conseillé : .mp3 (compatible partout).
Autres formats possibles (.ogg, .m4a) : renomme le fichier et adapte
l'attribut src de <audio id="bg-music"> dans index.html.

Astuce : un MP3 ~128 kbps suffit largement et reste léger à charger.
